
# MinaEdu - منصة الأساتذة (PWA)

تطبيق ويب تقدمي (Progressive Web App) متخصص لإدارة الدروس والمحتوى التربوي للأساتذة في الطورين المتوسط والثانوي.

## المميزات
*   **للأساتذة فقط**: بيئة عمل مخصصة للمهام البيداغوجية.
*   **Offline First**: يعمل بدون اتصال بالإنترنت.
*   **إدارة المذكرات**: تحضير الدروس، التعديل، الطباعة، والتصدير (Word/PDF).
*   **التخطيط السنوي**: توزيع الوحدات التعليمية على الأسابيع.
*   **دفتر التنقيط**: حساب المعدلات ومتابعة التلاميذ.
*   **الذكاء الاصطناعي**: مساعد ذكي لتوليد وضعيات انطلاق وأسئلة اختبارات.

## البنية التقنية (Architecture)

### الواجهة الأمامية (Frontend)
*   **Framework:** React + Vite
*   **Styling:** Tailwind CSS (RTL supported)
*   **State:** Context API + LocalStorage (Offline support) + IndexedDB (File storage)

### الواجهة الخلفية (Backend Schema)
تم تصميم النظام ليكون جاهزاً للربط مع **PostgreSQL** (عبر Supabase أو NestJS).
ملف `schema.sql` المرفق يحتوي على:
*   جداول المستخدمين (`users`) والأقسام (`classes`).
*   هيكلية مرنة للدروس والاختبارات باستخدام `JSONB`.
*   جداول للتخطيط السنوي وسجل المتابعة اليومي.

## التشغيل والتطوير

1.  تثبيت المكتبات:
    ```bash
    npm install
    ```

2.  تشغيل السيرفر المحلي:
    ```bash
    npm run dev
    ```

3.  بناء النسخة النهائية:
    ```bash
    npm run build
    ```
